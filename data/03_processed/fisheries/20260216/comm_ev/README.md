# Commercial Fisheries Exchange Values

## Overview

| Detail | Value |
|--------|-------|
| Description | Processed commercial fisheries exchange value (EV) data for the Main Hawaiian Islands (MHI) |
| Reporting unit | DAR catch areas (82 areas) |
| Temporal coverage | 1997--2021 |
| Source data | HDAR Commercial Marine Landings |
| Ecosystem attribution | Fish Species over Ecosystem Type (FSOET) dataset |
| EV definition | Gross Revenue - Marginal Costs, per the UN SEEA-EA framework |
| Prepared by | Alemarie Ceria |
| Processing date | 02/16/2026 |

## Files

| File | Format | Description |
|------|--------|-------------|
| `{YYYYMMDD}_tidied_comm_ev.csv` | CSV | Tabular data without geometry. Use for tables and line plots. |
| `{YYYYMMDD}_tidied_comm_ev.gpkg` | GeoPackage | Tabular data with DAR catch area geometries. Use for choropleth maps. |
| `{YYYYMMDD}_tidied_comm_ev.geojson` | GeoJSON | Tabular data with DAR catch area geometries. Use for choropleth maps. Natively readable by most web mapping libraries. |

## Species groups and ecosystem types

**Species groups:** Deep 7 Bottomfish, Shallow Bottomfish, Pelagics, Reef-Associated, All Species

**Ecosystem types:** Inshore -- Reef, Coastal -- Open Ocean, All Ecosystems

## Data dictionary

| Column | Type | Description |
|--------|------|-------------|
| `year` | integer | Calendar year |
| `area_id` | integer | DAR catch area identifier |
| `county` | character | County name |
| `county_olelo` | character | County name in 'Olelo Hawai'i (display only) |
| `species_group` | character | Species group |
| `ecosystem_type` | character | Ecosystem type attribution |
| `exchange_value` | numeric | Exchange value in USD |
| `exchange_value_formatted` | character | Currency-formatted exchange value (display only) |
| `geometry` | geometry | DAR catch area polygon (spatial files only) |

## Coordinate reference system

EPSG:4326 (WGS 84). Applies to `.gpkg` and `.geojson` files only.

## Dashboard usage

| Component | File to use | Key columns |
|-----------|-------------|-------------|
| Choropleth maps | `.gpkg` or `.geojson` | Fill: `exchange_value`. Filter by `year`, `species_group`, `ecosystem_type`, `county`. |
| Tables and line plots | `.csv` | Group or filter by `year`, `species_group`, `ecosystem_type`, `county`. |
