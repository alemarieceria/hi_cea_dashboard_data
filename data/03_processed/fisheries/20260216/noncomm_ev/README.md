# Non-Commercial Fisheries Exchange Values

## Overview

| Detail | Value |
|--------|-------|
| Description | Processed non-commercial fisheries exchange value (EV) data for the Main Hawaiian Islands (MHI) |
| Reporting unit | Islands (6 islands: Hawai'i, Kaua'i, Lana'i, Maui, Moloka'i, O'ahu) |
| Temporal coverage | 2005—2022 |
| Source data | MRIP Hawai'i Marine Recreational Fishing Survey |
| Ecosystem attribution | Fish Species over Ecosystem Type (FSOET) dataset |
| EV definition | Gross Revenue - Marginal Costs, per the UN SEEA-EA framework |
| Prepared by | Alemarie Ceria |
| Processing date | 02/16/2026 |

Ni'ihau and Kaho'olawe are not included because the MRIP survey does not cover those islands.

## Files

| File | Format | Description |
|------|--------|-------------|
| `{YYYYMMDD}_tidied_noncomm_ev.csv` | CSV | Tabular data without geometry. Use for tables and line plots. |
| `{YYYYMMDD}_tidied_noncomm_ev.gpkg` | GeoPackage | Tabular data with island geometries. Use for choropleth maps. |
| `{YYYYMMDD}_tidied_noncomm_ev.geojson` | GeoJSON | Tabular data with island geometries. Use for choropleth maps. Natively readable by most web mapping libraries. |

## Species groups and ecosystem types

**Species groups:** Herbivores

**Ecosystem types:** Inshore — Reef, Coastal — Open Ocean, All Ecosystems

All herbivore species map to 100% reef habitat in this iteration.

## Data dictionary

| Column | Type | Description |
|--------|------|-------------|
| `year` | integer | Calendar year |
| `island` | character | Island name |
| `island_olelo` | character | Island name in 'Olelo Hawai'i (display only) |
| `county` | character | County name |
| `county_olelo` | character | County name in 'Olelo Hawai'i (display only) |
| `species_group` | character | Species group (Herbivores) |
| `ecosystem_type` | character | Ecosystem type attribution |
| `exchange_value` | numeric | Exchange value in USD |
| `exchange_value_formatted` | character | Currency-formatted exchange value (display only) |
| `geometry` | geometry | Island polygon (spatial files only) |

## Coordinate reference system

EPSG:4326 (WGS 84). Applies to `.gpkg` and `.geojson` files only.

## Dashboard usage

| Component | File to use | Key columns |
|-----------|-------------|-------------|
| Choropleth maps | `.gpkg` or `.geojson` | Fill: `exchange_value`. Filter by `year`, `island`, `ecosystem_type`. |
| Tables and line plots | `.csv` | Group or filter by `year`, `island`, `ecosystem_type`, `county`. |
